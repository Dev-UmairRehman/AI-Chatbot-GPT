# 🤖 Intelligent Chatbot with NLP

An AI-powered chatbot built using **GPT-4 API**, designed to handle customer queries 24/7 with high accuracy and natural conversational flow. Built with a modern stack including **Python (Flask) and React**, this chatbot can be integrated into websites and applications for seamless customer support.

---

## 🚀 Overview

This project delivers an intelligent conversational interface that understands user intent and responds in real-time. It reduces the need for human support agents while improving response speed and user experience.

---

## 🔥 Features

* 💬 Natural language understanding using GPT-4 API
* ⚡ Real-time chat interface
* 🕐 24/7 automated customer support

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/intelligent-chatbot.git
cd intelligent-chatbot
```

### 2. Setup Backend (Flask)

```bash
cd backend
python -m venv venv
source venv/bin/activate   # On Windows: venv\\Scripts\\activate
pip install -r requirements.txt
```

Create `.env` file:

```env
OPENAI_API_KEY=your_api_key
```

Run backend:

```bash
python app.py
```

### 3. Setup Frontend (React)

```bash
cd frontend
npm install
npm start
```
